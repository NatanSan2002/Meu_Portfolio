Minhas Habilidades

Essas são as linguagens com que tive mais contato e prática.

HTML

•

É a marcação dos conteúdos do site. Mesmo sendo relativamente "simples" ele que é o responsável pela estrutura/organização das Tags, colocando-as em hieraquias que seram melhor estilizadas pelo CSS.

Repositório

CSS

•

É responsável pela estilização do site. Assim podemos dizer que o HTML e o CSS são uma dupla, enquanto o  HTML faz os "ingredientes do bolo"  o CSS organiza-os e também faz o recheio dele. 

Repositório

JavaScript

•

É uma linguagem de programação que também  é utilizada na web, junto com o HTML e CSS. No site, o JavaScript  é o que faz toda a execução dinâmica com sua lógica, podendo até interferir no HTML e CSS.

Repositório

PHP

•

É uma linguagem responsável por fazer a "ponte" entre o frontend e o backend, podendo por exemplo resgatar dados do servidor e exibi-los pro usuário  e vice-versa: pegar dados do usuário e envia-los para o servidor. 

Repositório

SQL

•

É uma linguagem de pesquisa e estrutura em tabelas, padrão para banco de dados relacional.  Na web é utilizada via a linguagem PHP.

Repositório

REACT JS

•

É um framework do JavaScript, seu principal foco é a interligação mais prática entre o JS e o HTML/CSS, assim um site que usa React geralmente possui recursos mais avançados com código mais limpo e organizado do que um que não.
